# 🌾 FarmKart - Farmer to Customer Marketplace

A full-stack e-commerce web application that connects farmers directly with customers. Users can browse products, add items to cart, and place orders seamlessly.

---

## 🚀 Features

- 🔐 User Authentication (Login & Register)
- 🛍️ Browse fresh farm products
- 🔍 Search functionality
- 🛒 Add to Cart / Remove items
- 📦 Order placement system
- 📍 Address & checkout flow
- 💳 Payment simulation
- 🖼️ Dynamic product images from database

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- JavaScript (ES6)
- HTML5, CSS3

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB Atlas
